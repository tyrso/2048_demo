# 2048Demo

## 遊戲介紹
遊戲初始會有一個4*4的盤面
每次操作可使方塊整體以上下左右其一方向移動。如果兩個帶有相同數字的方塊在移動中碰撞，則它們會合成為一個方塊，其所帶數字變為兩者之和。每次移動時，會有一個值為2或者4的新方塊出現(機率為0.9、0.1)，所出現的數字都是2的冪次，當合成時，即得到方塊分數，例:將兩個2048方塊合成為4096，則分數+4096分
